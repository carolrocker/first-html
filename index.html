Principais Conceitos

HTML

HTML é uma linguagem de marcação de hipertexto. Os hipertextos, por sua vez, são a junção de vários elementos — palavras, vídeos e conteúdos —, que, quando conectados, estabelecem uma rede de dados que permite a comunicação, o armazenamento e o compartilhamento de informações.

O HTML é utilizado na construção de páginas na Web, cujos textos delimitados por nomes podem ser interpretados pelos navegadores e cada um destes delimitadores possuem uma função e comportamento específico.

É importante destacar, entretanto, que o HTML não é uma linguagem de programação, mas sim um arquivo de marcação.

Clients

No desenvolvimento web, client ou cliente em português, refere-se ao usuário final de um aplicativo web. E tudo que é exibido ou ocorre no dispositivo deste usuário como textos, imagens e demais execuções que um aplicativo execute no navegador, acontece no "lado do cliente".

Além disso, cada "lado" possui uma linguagem de programação específica para o seu desenvolvimento.

No client-side, por exemplo, podemos utilizar o JavaScript.

Servers

O servidor é o componente do sistema que fornece os serviços ou recursos solicitados pelos clientes. Estes servidores podem ser um computador projetado especificamente para gerenciar e fornecer serviços aos clientes, ou ainda, um software que é executado em um servidor de rede.

Existem vários tipos de servidor, entre eles podemos citar os firewalls, streamings, servidores de e-mail e web.

Assim como o client-side, o server-side também possui linguagens de programação específicas para o seu desenvolvimento. Neste caso, as mais utilizadas são Java, Javascript, Ruby, entre outros.

HTML na prática

Principais tags

As tags são elementos usados formatar os conteúdos: textual ou visual em documentos HTML. Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada. As tags também podem possuir atributos, que nada mais são que palavras especiais que controlam o comportamento do elemento.

Abaixo confira as principais tags:

  • head: Armazena tudo aquilo que é pré-carregada antes da exibição ao usuário.
  • title: O título impresso na aba do navegador.
  • body: Tudo o que é exibido para o usuário.

Algumas tags para formatação do texto são:

  • h1 ao h6: Definem o tamanho do texto, sendo que h1 deixa o texto maior e vai diminuindo conforme o avanço dos números.
  • p: Cria um novo parágrafo.
  • hr: Adiciona uma linha de "separação" abaixo do texto.
  • blockquote: Adiciona uma citação.
  • strong: Deixa o texto em negrito.
  • i: Deixa o texto em itálico.
Confira outras tags no site do W3schools!

Listas

Existem 2 tipos de listas: As listas ordenadas (ol) e as não-ordenadas (ul). Tais listas são chamadas de "tag pai" e podem conter outras listas dentro delas, as List Item (li), também conhecidas como tag filha.

As listas ordenadas, como o próprio nome já diz, é organizada em ordem numérica. Já as não-ordenadas, embora ainda sigam uma ordem, não possuem um número indicando seus tópicos, mas sim marcadores.

Por exemplo:

    Lista ordenada:
  1. João
  2. Maria
  3. Carlos
    Lista não-ordenada:
  • João
  • Maria
  • Carlos

Links

Os links são amplamente utilizados ao navegarmos pelas páginas web, seja para sermos redirecionados para o mesmo ste, mas em diferentes abas ou até para sites externos.

Para acrescentar um link ao código HTML usa-se a tag "a href=". Você pode querer que este link abra na mesma página utilizando o "_self" ou em uma nova guia através do "_blank". Ambos são utilizados, porém cada um possui uma indicação ideal. Por exemplo, para manter o usuário no seu site, o mais indica é que links externos sejam abertos em uma nova guia, dessa forma, o cliente não perderá a sua página.

Outra possibilidade é criarmos um link para a mesma página, porém em outro tópico do site através da ancoragem.

Você pode observar o funcionamento da ancoragem neste site, onde os links te levam para diferentes pontos de uma mesma página.
